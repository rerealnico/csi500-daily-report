# 市场洞察（Market Insight） 问题分类参考

本文档列出金融分析工具支持的所有问题类别及示例，供 Agent 判断用户问题是否属于金融分析范畴。

---

## 一、个股分析

### 1. 整体判断类
| 子类         | 定义                                                  | 示例                                  |
| ------------ | ----------------------------------------------------- | ------------------------------------- |
| 综合分析     | 全面评估股票基础价值与走势（基本面+技术面+走势+估值） | 某某股票怎么样？帮我分析一下XX        |
| 投资价值分析 | 重点了解公司基本面+估值                               | 某某股票是否值得买？XXX有投资价值吗？ |

### 2. 基本面分析类
| 子类       | 定义                       | 示例                                |
| ---------- | -------------------------- | ----------------------------------- |
| 整体基本面 | 经营绩效、成长性、护城河等 | XXX基本面如何？请分析经营情况       |
| 基本信息   | 公司基本信息               | 主营业务是什么？主要产品？          |
| 业务结构   | 业务经营相关信息           | 收入/利润结构？地区/产品占比？      |
| 公司关系   | 股权、上下游关系           | 客户/供应商是谁？与某公司什么关系？ |
| 主题映射   | 公司与行业/概念的关系      | 和某行业有什么关系？属于某概念吗？  |
| 竞争优势   | 行业地位、市占率、护城河   | XXX行业地位如何？有什么壁垒？       |
| 财务质量   | 多维度财务指标             | 财务是否健康？负债结构如何？        |
| 增长前景   | 战略布局、未来发展         | 未来靠什么增长？发展前景？          |
| 公司治理   | 管理举措、内控、治理结构   | 评估信息披露制度                    |
| 分红       | 分红派息情况               | 最近几年分红怎么样？股息稳定吗？    |
| 股东结构   | 股东结构变化               | 谁是大股东？控股股东有没有变？      |
| 单指标趋势 | 具体指标时间序列变化       | 近年营业收入情况？利润增长？        |

### 3. 财报解读类
| 子类           | 定义               | 示例                      |
| -------------- | ------------------ | ------------------------- |
| 财报查询和解读 | 财报信息，业绩重点 | XXX半年报怎么样？分析财报 |

### 4. 事件分析类
| 子类           | 定义                 | 示例                           |
| -------------- | -------------------- | ------------------------------ |
| 公告事件分析   | 公告内容理解         | 近期公告解读？拟重组意味什么？ |
| 外部事件影响   | 事件到影响的传导关系 | XX事件对XX股票有什么影响？     |
| 市场消息判断   | 传闻验证和可能性分析 | 某公司是否会重组？合并概率？   |
| 信息和公告查询 | 信息查询             | 为什么停牌？什么时候复牌？     |

### 5. 股价与走势类
| 子类         | 定义                       | 示例                       |
| ------------ | -------------------------- | -------------------------- |
| 涨跌归因     | 消息面、技术面和资金面分析 | 为什么涨？为什么跌？       |
| 技术分析     | 技术面重点分析             | 帮我分析技术面             |
| 资金面分析   | 资金流向和结构             | 资金流向情况？             |
| 走势分析     | 分析过去走势               | 今天怎么样？近期走势如何？ |
| 短期走势预测 | 预测短期走势               | 最近会反弹吗？明天能涨吗？ |
| 走势预测     | 综合预测未来走势           | 后市如何走？还能涨吗？     |
| 事件催化     | 事件对股价的影响           | XX事件后明天会涨吗？       |

### 6. 外部观点类
| 子类     | 定义                   | 示例                      |
| -------- | ---------------------- | ------------------------- |
| 机构观点 | 评级、目标价、一致预期 | 机构怎么说？评级/目标价？ |

### 7. 估值类
| 子类         | 定义              | 示例                       |
| ------------ | ----------------- | -------------------------- |
| 历史估值对比 | 估值高估/低估判断 | 目前估值如何？现在低估吗？ |

## 二、大盘分析（A股、港美股）

### 1. 点位与走势类
| 子类       | 定义                   | 示例                         |
| ---------- | ---------------------- | ---------------------------- |
| 走势预测   | 大盘走势和关键指标     | 大盘后续走势？能突破XX点吗？ |
| 走势分析   | 走势、量价             | 今天大盘怎么样？成交量？     |
| 资金面分析 | 资金流入流出和结构     | 大盘资金流向？               |
| 涨跌归因   | 消息面、技术面、资金面 | 大盘为什么下跌/上涨？        |
| 事件催化   | 事件到走势传导         | 根据XXX事件判断明天走势      |
| 热门板块   | 主要涨跌板块           | 今天哪些板块在涨？           |

### 2. 事件分析类
| 子类     | 定义               | 示例                             |
| -------- | ------------------ | -------------------------------- |
| 信息查询 | 市场消息           | 今天有什么消息？A股重要消息？    |
| 事件影响 | 宏观政策对大盘影响 | 美联储议息结果？政治局会议分析？ |

### 3. 估值类
| 子类         | 定义         | 示例                       |
| ------------ | ------------ | -------------------------- |
| 历史估值对比 | 大盘估值分位 | 当前大盘处于什么估值水平？ |

## 三、概念/板块/行业/ETF 分析

| 子类     | 定义                    | 示例                            |
| -------- | ----------------------- | ------------------------------- |

| 投资价值 | 基本面+估值             | XX行业有价值吗？值不值得买？    |

### 2. 基本面分析类
| 子类     | 定义                     | 示例                          |
| -------- | ------------------------ | ----------------------------- |
| 增长前景 | 发展前景                 | XX行业前景？芯片ETF未来趋势？ |

### 3. 事件分析类
| 子类         | 定义         | 示例                   |
| ------------ | ------------ | ---------------------- |
| 外部事件影响 | 事件传导逻辑 | XX事件对XX行业影响？   |
| 信息查询     | 相关领域新闻 | XX板块最近有什么新闻？ |

### 4. 点位与走势类
| 子类         | 定义                 | 示例                       |
| ------------ | -------------------- | -------------------------- |
| 涨跌归因     | 消息面+技术面+资金面 | XX行业/ETF为什么大跌？     |
| 资金面分析   | 资金流入流出和结构   | XX行业近期资金流向？       |
| 技术分析     | 技术面重点分析       | 分析XX行业技术面           |
| 走势分析     | 走势、量价           | XX行业近期走势？           |
| 短期走势预测 | 短期走势预判         | XX板块明天能涨吗？         |
| 走势预测     | 综合未来走势预测     | XX板块后市如何？           |
| 事件催化     | 事件传导逻辑         | XX事件后明天XX板块会涨吗？ |

### 5. 外部观点类
| 子类     | 定义         | 示例             |
| -------- | ------------ | ---------------- |
| 机构观点 | 研报观点总结 | XX板块研报观点？ |

### 6. 估值类
| 子类         | 定义         | 示例             |
| ------------ | ------------ | ---------------- |
| 历史估值对比 | 板块估值分位 | XX行业估值高吗？ |

## 五、金融资讯查询

### 1. 个股资讯类
| 子类           | 定义                           | 示例                                                         |
| -------------- | ------------------------------ | ------------------------------------------------------------ |
| 个股新闻检索   | 查询单只股票的近期新闻资讯     | 长江电力最近有什么新闻？格力电器最近有哪些资讯？             |
| 利好利空分类   | 按利好/利空对个股消息分类整理  | 整理比亚迪最近的利好利空消息                                 |
| 限时资讯汇总   | 指定时间范围内的个股资讯整理   | 整理长江电力最近48小时的资讯                                 |
| 资讯简讯生成   | 将资讯精简汇总并附带来源链接   | 整理伊利股份的近期资讯简讯并提供链接                         |

### 2. 多股资讯类
| 子类               | 定义                             | 示例                                                                       |
| ------------------ | -------------------------------- | -------------------------------------------------------------------------- |
| 多股资讯批量检索   | 同时查询多只股票的近期资讯       | 整理长江电力、中国移动、五粮液最近的资讯                                   |
| 多股利好利空整理   | 多只股票的消息按利好利空分类整理 | 根据利好和利空分类整理格力电器、伊利股份、海尔智家相关的资讯               |
| 多股限时资讯汇总   | 多股在指定时间范围内的资讯汇总   | 分类整理48小时内长江电力、中国移动、五粮液相关的资讯，整理成简讯并提供链接 |

### 3. 板块/行业资讯类
| 子类           | 定义                       | 示例                                       |
| -------------- | -------------------------- | ------------------------------------------ |
| 板块新闻聚合   | 特定板块或行业的资讯汇总   | 半导体板块最近有什么重要新闻？             |
| 行业动态       | 行业层面的政策/趋势性新闻  | 新能源汽车行业最近有哪些政策动态？         |

### 4. 市场资讯类
| 子类           | 定义                           | 示例                                   |
| -------------- | ------------------------------ | -------------------------------------- |
| 市场热点新闻   | 整体市场的重要新闻和热点       | A股今天有哪些重要资讯？               |
| 政策与宏观资讯 | 影响市场的宏观政策/经济资讯    | 最近有什么影响股市的政策消息？         |
