#### 🔵 P2 — `levels` 循环永远不执行（逻辑死代码）

```javascript
var levels = [mn, mid, mx];
for (var li = 0; li < levels.length; li++) {{
    var lv = levels[li];
    if (lv === mn || lv === mx || Math.abs(lv-mid) < rg*0.05) continue;  // ← 三个值全部被跳过
    ...
}}
```

- `lv = mn` → 被 `lv === mn` 跳过
- `lv = mid` → 被 `Math.abs(mid-mid) = 0 < rg*0.05` 跳过
- `lv = mx` → 被 `lv === mx` 跳过

**结果：** 这个 `for` 循环的循环体永远不会执行，是死代码。虽然不影响功能（因为后面还有 5 条等分线覆盖了这些位置），但冗余代码增加阅读负担。
